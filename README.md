## NestJs Install Dependencies

```bash
$ npm install
```

## Run NestJs (Default Port is 3000)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## React Install Dependencies

```bash
$ npm install
```

## Run React App (Default Port is 4000)

```
npm run dev
```
